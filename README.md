# Spotify Account Changer
Easily switch from an account to another with this tool

![](https://cdn.discordapp.com/attachments/923633338378489856/1027552747924508752/banner.png)

## ⚠️ Disclaimer
🔧 Spotify Account Changer (AKA S.A.C. or SAC) is a tool that helps Spotify users changing between accounts, without inserting credentials everytime, especially for strong password users that have to open their password manager.

🙅‍♂️ SAC doesn't allows the user to generate/create, delete nor edit Spotify server-sided accounts.

💻 SAC doesn't store anything remotely, everything is done locally on user's pc, and it doesn't apply any edits to user's Spotify account/profile.

👮‍ For any security check, this tool is free and open source so it can be revised and edited to assure the user that this tool is completely harmless.

## 📖 Requirements
- Windows
- Spotify
- [OPTIONAL] [Soggfy](https://github.com/Rafiuth/Soggfy)

![](https://cdn.discordapp.com/attachments/923633338378489856/1044367579738816622/Immagine_2022-11-21_223735.png)

## 🏃 Running the script
▶️ Start by opening Spotify, login with your desired account, ✅ check the Remember Me option then customize settings if you want to

[![](https://cdn.discordapp.com/attachments/923633338378489856/1034077280814764133/dlbutton.png)](https://raw.githack.com/Zoult/spotify-account-changer/main/sac.bat)

📃 Press `t` to **list** saved accounts

💾 Press `s` to **save** an account (spaces not allowed, use dashes or points or whatever you want)

✏️ Press `r` to **rename** an account (spaces not allowed, use dashes or points or whatever you want)

🗑️ Press `d` to **delete** an account

📥 Press `l` to **load** an account

🥚 Press `g` to **load** an account opening Soggfy (remember to edit the <i>`soggfy` value</i> at the 4th line with the path you downloaded Soggfy)

📤 Press `u` to **unload** the current account
